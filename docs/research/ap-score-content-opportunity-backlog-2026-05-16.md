## 1. Executive decision

Semrush Keyword Gap shows that the biggest near-term opportunity is **not** a new-site build, broad redesign, or monetization push. The site is already live; competitor analysis is being used to find unmet search demand for SEO growth. The strongest move is to turn the current 16-page static calculator cluster into a clearer, exact-match, internally linked AP calculator product.

### Priority order

1. **P0 — Fix and strengthen existing high-demand calculator pages/modules**
   - Hub and subject pages already exist, but Semrush still sees us missing core terms where competitors rank.
   - First work should improve titles/H1s, intro copy, calculator-first UX, conversion-table explanations, target-score guidance, FAQ/schema, and internal links.

2. **P0 — Improve weak pages that already get GSC impressions**
   - AP Gov and AP Psychology are already visible in GSC and also appear in Semrush weak keywords.
   - These should be the fastest feedback loop: indexed/visible pages + known weak keyword set.

3. **P1 — Create one supporting AP score-distribution page/module**
   - `ap score distribution 2025` and `ap score distribution` are high-volume missing opportunities.
   - Build as an informational support page only if source data is cited and disclaimers are clear.

4. **P1 — Add AP exam schedule / score release support content**
   - `ap exam schedule`, `ap test schedule`, and score-release queries have large volume and reasonable KD.
   - These should be lightweight support modules/pages linked from the hub, not allowed to distract from calculator intent.

5. **P2 / Watchlist — Broad grade/test calculator and SAT/ACT/practice-test clusters**
   - Very high volume but mostly off-scope for the AP score calculator product.
   - Do not build now unless Ningge chooses a second product line.
